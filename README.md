This project helps people who need blood find donors and blood banks nearby in a safe and trustworthy way. It makes sure all blood requests are real, so no one gets tricked. It also helps blood donors connect with those who need blood, making it easier and faster to save lives.

The backend, which handles all the data and rules, is built using **Java Spring Boot**. The frontend, which is what users see and use, is made with **HTML, CSS, and JavaScript**. All the information like users and blood requests are stored safely using **SQL**, a tool that helps manage databases.
